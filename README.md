COJ - Caribbean Online Judge (Web API)
==============================

### Environment variables
#### DATABASE_HOST
Database host
#### DATABASE_NAME
Database name
#### DATABASE_PASSWORD
Database password
#### DATABASE_USERNAME
Database username
#### SMTP_HOST
Smtp server host
#### SMTP_PASSWORD
Smtp server password
#### SMTP_PORT
Smtp server port
#### SMTP_USERNAME
#### SMTP_FROM
From Address used when email are sent
#### JWT_SECRET
Secret used to encode JWT.
